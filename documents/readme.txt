Man Vs Machine is a king of the hill game that pits players against AIs to fight over an objective point
Players play as a tank and move across a side scrolling map to find and attack enemies
To run the game, simply execute the main file in a folder with all the other files. Press space bar on the landing screen for indepth in game tutorial
The only library required is the CMU graphics package 
List of useful shortcut keys: 
G: godmode in game
L: instant Lose
K: instant Win
